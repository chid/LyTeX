Introduction

LyTeX is a portable TeX suite for Windows, including TeXLive, TeXworks and LyX.

You can download latest LyTeX from http://code.google.com/p/lytex/.

Questions

Quesiton 1: How to install or update TeXLive packages?

Answer 1: Run Setup.exe and select "Manage or update TeX programs and packages".

Quesiton 2: How to update LyX to latest version?

Answer 2: Run Setup.exe and select "Update to latest LyX from internet".

Quesiton 3: How to add spell checking for other languages in TeXworks?

Answer 3: Download dictionaries from here. Assume you have downloaded dictionaries-el.oxt file. Rename it to dictionaries-el.zip and extract it. Put all *.dic and *.aff files into TeXLive\tlpkg\texworks\dictionaries directory and restart TeXworks.

Quesiton 4: How to add spell checking for other languages in LyX?

Answer 4: Download dictionaries from here. Then install it and remember where it was installed (e.g. C:\Documents and Settings\All Users\Application Data\Aspell\Dictionaries). Open that directory and copy all files except Uninstall-AspellDict-*.exe to LyX\aspell\Dictionaries directory. Restart LyX and it's OK now.


English

基本介绍

LyTeX 是一个绿色小巧的 TeX 套装，它包含了 TeXLive，TeXworks 和 LyX。

LyTeX 的更新和反馈页面在这里：http://bbs.ctex.org/viewthread.php?tid=46857。

问题解答

问题 1: 如何安装或者更新 TeXLive 宏包？

解答 1: 运行 Setup.exe 并选择“升级或管理 TeX 程序和宏包”。

问题 2: 如何升级 LyX 到最新版本？

解答 2: 运行 Setup.exe 并选择“在线升级 LyX 到最新版本”。

问题 3: 如何在 TeXworks 程序中添加其它语言的拼写检查？

解答 3: 从这里下载所需的词典文件。假定你下载了 dictionaries-el.oxt 文件。将它改名为 dictionaries-el.zip 然后解压它. 将所有的 *.dic and *.aff 文件放到 TeXLive\tlpkg\texworks\dictionaries 目录中，重新打开 TeXworks 即可。

问题 4: 如何在 LyX 程序中添加其它语言的拼写检查？

解答 4: 从这里下载所需的词典文件。然后安装它并记住它所安装到的目录位置（例如 C:\Documents and Settings\All Users\Application Data\Aspell\Dictionaries）。打开该目录，复制除了 Uninstall-AspellDict-*.exe 文件之外的其它所有文件到 LyX\aspell\Dictionaries 目录中. 重新打开 LyX 即可。
